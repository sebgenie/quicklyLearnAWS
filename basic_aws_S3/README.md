## First execution (on linux (Ubuntu, debiant) or MacOS)

1- Install the trendencies by running the `script.sh` of the root directory
e.g: sh script.sh 

2- Enter in your exercise work directory
    
    e.g: cd basic_aws_S3/

3- Create a S3 bucket and make it public.
